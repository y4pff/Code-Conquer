Recipe Platform

A full-stack web application where users can create, share, and review recipes.

Live Site:

https://code-conquer-mocha.vercel.app

Features:
User registration and login
Create, edit, and delete recipes
Leave ratings and comments
Save favourite recipes
View all recipes with filtering and sorting
Tech Stack:
Frontend: React (Vite)
Backend: Supabase (PostgreSQL + Auth)
Deployment: Vercel
Run Local:
git clone https://github.com/y4pff/Code-Conquer.git
cd recipe-platform
npm install
npm run dev

Create a .env file:

VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
